EFFECTS IMPLEMENTED
────────────────────
1. Time Update Effect
   - Updates every 1 second
   - Keeps watch display current

2. Heart Rate Increase Effect
   - Triggered when button is held
   - Increases by 2 BPM every 200ms
   - Caps at 150 BPM

3. Auto-Normalize Effect
   - Triggered after 30 seconds of inactivity
   - Gradually decreases heart rate
   - Returns to 72 BPM

KEY FUNCTIONS
───────────────
1. increaseHeartRate()
   - Sets heartRateIncreasing to true
   - Starts interval for gradual increase

2. normalizeHeartRate()
   - Starts interval for gradual decrease
   - Resets abnormal state when complete

3. resetSimulation()
   - Clears all intervals and timeouts
   - Resets all data to initial state

================================================================================
HOW TO USE
================================================================================

ACCESSING THE FEATURE
──────────────────────
1. Log in to SafeTourAI user dashboard
2. Look for "Virtual Watch" in the left sidebar
3. Click it to open the simulator

BASIC OPERATION
─────────────────
1. INCREASE HEART RATE
   - Hold down the red "Increase Heart Rate" button
   - Heart rate gradually increases (2 BPM every 200ms)
   - Maximum reaches 150 BPM
   - Release to stop increasing

2. NORMALIZE HEART RATE
   - Click the green "Normalize" button
   - Heart rate gradually decreases back to 72 BPM
   - Takes about 30-40 seconds to fully normalize

3. AUTO-NORMALIZE
   - Stop clicking the increase button
   - System automatically normalizes after 30 seconds
   - No manual action needed

4. RESET EVERYTHING
   - Click "Reset Simulation" button
   - All data returns to initial state
   - Heart rate: 72 BPM
   - Steps: 8,547
   - Battery: 85%
